# Final Project - Portfolio

## 개발환경

- React 18.3.1 + Typescript with Vite
- React Router
- Recoil + Tanstack Query v5
- SCSS + Framer Motion
- Supabase

## 요구사항

- Memoization 활용
- useTransition 활용
- React.lazy와 Suspense 활용
- Composition Component

- 상태정의

- 공통 컴포넌트 분류
